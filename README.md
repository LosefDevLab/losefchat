## 部署极其方便

##### 你只需要输入2，然后再输入一个端口号就能启动服务器

##### 如果有需要，还可以配置内网穿透（如果你的电脑本来就是公网环境，则不用）

##### 并且我们还有一个非常好...简易的命令控制，只需要输入一小串文本，就能对服务器进行操作

##### 对于客户端呢，只需要输入1，然后输入IP和端口后就能连接服务器，然后尽情的和别人聊天

##### 如果您想友好的下线，建议您输入exit，会自动发送一串“我下线了拜拜”！

##### （我敢说没人使用上述功能，因为他们都是直接点控制台的关闭按钮的）

##### 还有，MODAREA能帮助你添加模组，只需要拉取LosefChat的代码，然后将模组代码复制过来添加到模组开发区域

##### 再在模组运行区域添加安装模组的代码，然后再次编译LosefChat，你就能使用第三方的功能了

## Windows&Linux 如何编译？

##### 请下载dotnet还有git（VERSION>=5.0）

```bash
git clone https://github.com/LosefDevLab/losefchat
cd losefchat
dotnet build
cd bin
cd Debug
#在这里，请cd到neta.b,其中a.b是你的dotnet的版本
./losefchat

```
